A core principal of how electronics work is due to the properties the electrons flow through. The first component we will discuss is the Capacitor

Capacitors

Formerely known as condensers are electronic components with the sole purpouse of storing electrostatic current. As apposed to the resistor, which releases energy, usually as heat. They are useful for blocking direct current(DC) while letting Alternating current (AC) to pass through. This propertey can be used for tuning radios and stabilizing power systems output. Electrolytic capacitors, are polarity sensitive( pictured in the top right corner of the image). The side with the black band is the negative side. These are called Electrolytic capacitors.


Integrated Circuts

Integrated Circuts, or IC are used to provide and easy interface for creating circuts, to make the makers life easier, similar in concept to the use of APIs by programmers for interfacing with common uses. ICs are basically compressed circuts, manufactured by machines for use of other circuts in smaller form. Contained within an IC, is an extremely small circut, usually for one purpouse or application. Common ICs are the LM386, a 1 Watt audio amplifer system on a chip, and the 555 Astable timer(NE555). These IC still require some construction, in parameters and outputs. They can be used to make almost all applications for simplification. Common uses are for timers, radio recievers and transmitters, or even simple gaming consoles, or computers, using the z80 or 6502 found in common game consoles of the 1980s to early 1990s. More advanced ARM b based microcontrollers, and single board packages exist as well.

Resistors can be described the component that limits flow in an electronic circuit. The unit of mesure is in Ohms. A multimeter or using the chart above, one can determine the value of the resistor. In series they can be used to lower voltage and prevent short circuits(a loop to the battery). Resistors are passive components that use material to add resistance to current flow. In addition, they can be used to lower voltage levels and reduce curent flow. There are four color bands on a resistor. Each color represents a value measured in Ohms. To identify the resistance value by hand, identify a the first band, on the outer edge of the resistor. The color represents a value on the chart above. An example is if the first color is red then the first significant number is 2. The second band represents the second digit in the resistor. For an example if the second value is orange and the first band is Red, then the resistance is 23. The third band is a multiplier that determines how much the value is to be multiplied. Our example resistor (Red-Orange-Black-Gold) would have a resistance of 230 Ohms. Lastly, the tolerance of the resistor is detmerined by the fourth and final band. It is typically gold, silver or not present at all. In our example, a gold tolerance is +/-5% so 11.5+/- varriance from the stated resistance. So the range of the resistor would be 218.5 to 241.5 Ohms. Do not forget, when designing or implementing resistors to take this factor in account. Causes of this varriance is temperature, humidity and other atmospheric conditions. This varriance can be reduced by soldering directly to the board instead of breadboarding and using temperature controlled Phase-Loop Locks.
